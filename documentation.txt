Bagi user :
- web SIG pariwisata ini dapat menampilkan 4 fitur, yaitu objek wisata, penginapan (hotel), oleh-oleh khas daerah, dan event yg di adakan di kota tersebut. Jdi ke 4 fitur ini dapat membantu user untuk pergi menuju ke tempat yang di inginkannya dengan rute terdekat yang sudah di tentukan dengan metode algoritma dijkstra agar user tidak perlu menempuh jarak yang jauh lagi.
- Kemudian ketika user ingin menuju ke objek wisata ataupun penginapan dan oleh² khas daerah (untuk event itu update krna bisa berubah²) dari titik user berada ke tempat tujuan yg ingin di tuju maka titik yg di tujunya itu juga akan memunculkan rekomendasi seperti contoh user ingin ke objek salah satu wisata maka akan muncul rekomendasi tmpat penginapan terdekat, oleh² khas daerah dan event(jika di adakan) ataupun sebaliknya begitu jga.
- Trus user juga dapat seperti user bisa menambah titik lokasi yang belum terdaftar di SIG tersebut (kalau bisa mas nya)

Bagi admin :
- untuk admin mungkin mas nya sudah tau ya seperti bagian crud dll dan bisa menentukan rute terdekat dari titik ke titik lain beserta yg lainnya.
